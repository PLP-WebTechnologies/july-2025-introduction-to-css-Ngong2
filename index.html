CSS Box Model in Action

This page demonstrates the fundamental concepts of CSS styling and the Box Model. Notice how margins, padding, and borders create spacing and visual hierarchy.

Understanding the Box Model

Content Area

This is the actual content of the element. The width and height properties define the size of this area.

Padding

Padding creates space between the content and the border. It's inside the element and takes on the background color.

Border

The border wraps around the padding and content. You can style its width, color, and style properties.

Margin

Margin creates space outside the border, separating this element from other elements. It's always transparent.
